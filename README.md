# Fixed Income Analysis

A Python-based fixed income project covering bond pricing, spot and forward rates,
duration, convexity, and hedging strategies using real ECB yield curve data.

## Project Structure
```
├── data/                  # ECB spot rates dataset (CSV)
├── notebook.ipynb         # Main Jupyter Notebook with all computations
└── README.md
```

## Data Source

European Central Bank — [Euro Area Yield Curves](https://www.ecb.europa.eu/stats/financial_markets_and_interest_rates/euro_area_yield_curves/html/index.en.html)

## Requirements
```bash
pip install numpy pandas matplotlib jupyter
```
